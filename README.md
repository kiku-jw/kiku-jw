## Current Focus

- Browser-local AI and file tools: practical utilities that run on user-provided files with clear previews and export paths.
- AI-assisted workflow automation: research, triage, document/data processing, and operator-facing tools.
- Durable agent workflows: GitHub control loops, Codex skills, task bundles, and small local systems that make AI work auditable.
- Small public utilities: focused tools with a concrete job, not broad platforms.

Private support, research, and operations repositories are intentionally omitted from this profile.
